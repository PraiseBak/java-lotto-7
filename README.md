# java-lotto-precourse

# 요구사항 명세
## 입력
- 유저에게 금액을 입력받는다
- 유저에게 구매 결과를 출력한다
- 유저에게 당첨 번호를 입력받는다
- 유저에게 보너스 번호를 입력받는다
- 유저에게 당첨 통계를 출력한다

## 로또
- 로또 번호는 1 - 45
- 중복되지 않는 6개의 숫자 
- 1등부터 5등까지
- 보너스 번호는 일반번호와 중복되지 않게
- 구입 금액에 해당하는 만큼 로또 발행
- 로또 1장당 1000원
  - 딱떨어지지않으면 안됨
- 당첨 번호, 보너스 번호 존재
- 당첨 내역 및 수익률 출격
- [ERROR] 시작 에러 메시지
